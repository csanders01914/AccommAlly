## How to Run

```bash
# Apply schema changes (if any)
npx prisma generate
npx prisma db push

# Seed the database
npx prisma db seed
```

**Warning:** The seed wipes the entire database before inserting. Never run in production.

## Portal Registration

New claimants can self-register at `/portal/register` using the organization code above.
Existing claimants (pre-seeded above) already have accounts and can log in directly at `/portal/login`.
